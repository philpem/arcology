Reclaiming space

Physical bytes are deduplicated automatically as content is hashed. To link artefacts hashed after a migration and prune legacy duplicate objects from storage, run on the web container: 

flask backfill-blobs            # link newly-hashed artefacts to blobs
flask dedup-artefacts           # report legacy non-canonical objects
flask dedup-artefacts --apply   # delete them and reclaim space
