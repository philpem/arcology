About Arcology

Arcology is a digital artefact catalogue for retrocomputing collections.

Track disk images (Supercard Pro, ImageDisk, ISO, etc.), scanned documents, and other digital files associated with your collection.

Features

  • Catalogue items with multiple digital artefacts
  • Link to external collection systems (Koillection, Collective Access, etc.)
  • Browse extracted file listings from disk images
  • Identify known files using hash databases (NSRL, custom)
  • Queue analysis jobs for automated processing
  • REST API for integration with analysis tools

API

The API is available at /api/. See the README for documentation.
