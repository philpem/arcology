## Quick Reference

| I want to… | Go to… |
|------------|--------|
| Add a new item to the collection | **Items** → New Item |
| Upload a disc image or file | Open the item, then **Upload Artefact** |
| See what's in a disc image | Open the artefact, scroll to **File Listing** |
| Find a file by MD5 hash | **Search** → `md5:abcdef1234…` |
| Re-run a failed analysis | Open the artefact, click **Re-analyse** |
| Create an API key | **Profile** (top right) → Generate API Key |
| Manage platforms and categories | **Taxonomy** menu |
